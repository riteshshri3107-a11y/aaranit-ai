Study Material — KidsBlock Sensor Guide

3.1 LED Blink

Component: White LED Module

LED (Light Emitting Diode) converts electrical energy into light. It has a long pin (+/anode) and a short pin (-/cathode).

How it works: Set a digital pin HIGH to turn LED on, LOW to turn off. Add delays between to create blinking effect.

Key concepts: Digital output, HIGH/LOW states, delay timing, GPIO pins

Pin: Digital pin (e.g., GPIO 2, 4, 5, 13, 15)

Try: Change delay time to make LED blink faster or slower. Use loops for patterns like SOS!

3.2 Sound Sensor

Component: Microphone + LM358 amplifier

Captures sound waves and converts to electrical signal. The LM358 amplifies weak signals for clear reading.

How it works: Microphone picks up sound → LM358 amplifies → ADC reads analog value (0-4095)

Key concepts: Analog input, ADC (Analog-to-Digital Conversion), signal amplification

Range: 0 (silent) to 4095 (very loud)

Try: Make a voice-activated fan — when sound > threshold, turn on motor!

3.3 PIR Motion Sensor

Component: RE200B-P pyroelectric sensor

Detects infrared radiation from warm moving objects (humans, animals). Based on pyroelectric effect.

How it works: All warm bodies emit infrared. When a person moves, the sensor detects changes in IR and outputs HIGH.

Key concepts: Digital input, infrared, pyroelectric effect, motion detection

Output: HIGH (1) = motion detected, LOW (0) = no motion

Try: Build a smart hallway light that turns on when someone walks by!

3.4 Passive Buzzer / Power Amplifier

Component: 8002b power amplifier + passive buzzer

Unlike active buzzers, passive buzzers need external pulse frequencies to produce sound. Different frequencies = different notes!

How it works: PWM signals at specific frequencies create musical notes: C=262Hz, D=294Hz, E=330Hz, F=349Hz, G=392Hz, A=440Hz, B=494Hz

Key concepts: PWM (Pulse Width Modulation), frequency, musical notes, tone generation

Try: Play a melody! Chain multiple tones with different beats to make music.

3.5 Photoresistor

Component: Light Dependent Resistor (LDR)

Resistance changes with light intensity — more light = lower resistance = higher analog reading.

How it works: Photons hit the semiconductor material, freeing electrons and reducing resistance.

Key concepts: Analog input, resistance, voltage divider, light measurement

Range: 0 (dark) to 4095 (bright light)

Try: Build a smart desk lamp that automatically adjusts brightness based on ambient light!

3.6 AD Button

Component: Analog-Digital Button Module

Multiple buttons on one analog pin! Each button connects a different resistor, producing a unique analog value.

How it works: Voltage divider with different resistors. Each button press produces a distinct analog reading.

Key concepts: Analog input, voltage divider, resistor networks, multiplexing

Try: Use different buttons to control different LEDs or play different tones!

3.7 Ultrasonic Sensor

Component: HC-SR04 Ultrasonic Distance Sensor

Measures distance using sound waves. Sends ultrasonic pulse, times the echo, calculates distance.

How it works: Trigger pin sends 40kHz pulse → sound bounces off object → Echo pin receives it. Distance = time × speed of sound / 2

Key concepts: Speed of sound (343 m/s), echo timing, pulse measurement

Range: 2cm to 400cm (accuracy ±3mm)

Try: Build a parking sensor with buzzer — beep faster as objects get closer!

3.8 RFID Sensor

Component: MFRC522 RFID Reader/Writer

Reads unique ID from RFID cards and tags using radio frequency. Each card has a unique UID.

How it works: Reader emits radio waves → RFID tag draws power from field → tag transmits its UID back

Key concepts: Radio frequency, UID (Unique Identifier), SPI communication, contactless reading

Frequency: 13.56 MHz

Try: Build a door lock that only opens when the correct RFID card is scanned!

3.9 Servo Motor

Component: SG90 Micro Servo (180°)

Rotates to a precise angle (0-180°). Used in robot arms, locks, gates, and steering mechanisms.

How it works: PWM signal controls angle. Pulse width: 0.5ms=0°, 1.5ms=90°, 2.5ms=180°

Key concepts: PWM control, angle positioning, duty cycle, mechanical actuation

Range: 0° to 180°

Try: Build a robot arm or a smart door lock controlled by RFID!

3.10 Temperature & Humidity Sensor

Component: AHT20 Digital Sensor (I2C)

Measures both temperature and humidity with high precision using a 20-bit ADC.

How it works: Capacitive humidity sensor + bandgap temperature sensor. Communicates via I2C protocol.

Key concepts: I2C communication, digital sensor, calibration, environmental monitoring

Accuracy: Temperature ±0.3°C, Humidity ±2%RH

Try: Build a mini weather station displaying real-time temp and humidity on OLED!

3.11 Pressure Sensor

Component: BMP388 Barometric Pressure Sensor

Measures atmospheric pressure and can estimate altitude. Small, low power, high performance.

How it works: MEMS piezoresistive sensor detects air pressure changes. Altitude calculated from pressure difference.

Key concepts: Atmospheric pressure, altitude estimation, MEMS technology, I2C

Range: 300-1250 hPa (accuracy ±0.5 hPa)

Try: Calculate your floor level! Pressure drops ~1.2 hPa per 10m elevation.

3.12 Geomagnetic / Compass Sensor

Component: AK8975C 3-axis Electronic Compass

Detects Earth's magnetic field on X, Y, Z axes. Can determine compass heading (N/S/E/W).

How it works: Hall effect elements detect magnetic flux. 13-bit output for each axis.

Key concepts: Earth's magnetic field, Hall effect, compass heading, 3-axis measurement

Output: Heading 0-360° (0°=North, 90°=East, 180°=South, 270°=West)

Try: Build a digital compass that shows direction on the OLED display!

3.13 Fan / DC Motor

Component: DC Motor with Fan Blade

Controls a DC motor that can spin forward, reverse, or stop. Speed controlled by PWM.

How it works: Two control pins (MA, MB) determine direction. PWM duty cycle controls speed (0-255).

Key concepts: DC motor, H-bridge, PWM speed control, direction control

Modes: Forward, Reverse, Stop (gradual), Brake (immediate)

Try: Build a smart fan that turns on when temperature exceeds a threshold!

3.14 Servo Motor (270°)

Component: 270° Range Servo Motor

Extended range servo with 270° rotation. More range than standard 180° servo.

How it works: Same PWM principle as 180° servo but with extended mechanical range.

Key concepts: Extended rotation, precision positioning, PWM control

Range: 0° to 270°

Try: Use with a potentiometer to create a manual position controller!

3.15 WS2812 RGB LED Strip

Component: SK6812/WS2812 Addressable RGB LEDs (12 beads)

Each LED can show any color independently! Only needs one data pin for all LEDs.

How it works: Each LED has built-in controller. Data cascades through chain — each LED reads its color and passes the rest along.

Key concepts: Addressable LEDs, RGB color mixing, serial data protocol, daisy-chaining

Colors: 16.7 million colors (256 × 256 × 256 RGB)

Try: Create a rainbow effect by cycling colors through all 12 LEDs!

3.16 OLED Display

Component: 0.96" SSD1306 OLED Display (128×64 pixels)

Small but crisp display showing text, numbers, and simple graphics. No backlight needed — each pixel emits light!

How it works: Organic LEDs emit light when current passes through. I2C communication. Each pixel individually controlled.

Key concepts: I2C display, pixel graphics, text rendering, OLED technology

Resolution: 128×64 pixels, monochrome

Try: Display sensor readings in real-time — temperature, humidity, distance!

3.17 IR Receiver

Component: VS1838B IR Receiver Module

Receives infrared signals from remote controls. Each button sends a unique hex code.

How it works: IR receiver demodulates 38kHz carrier signal, decodes NEC protocol data.

Key concepts: Infrared communication, NEC protocol, carrier frequency, remote control

Frequency: 38 kHz carrier

Try: Use a TV remote to control your robot — each button triggers a different action!

3.18 IR Emitter

Component: IR LED Transmitter

Sends infrared signals to control TVs, air conditioners, or other IR devices.

How it works: IR LED pulses at 38kHz carrier frequency, encoding data in the pulse pattern.

Key concepts: IR transmission, signal encoding, carrier wave modulation

Try: Build a universal remote! Record IR codes with receiver, replay with emitter.

3.19 Joystick Module

Component: 2-axis Analog Joystick with Button

Two potentiometers (X, Y) and a pushbutton. Perfect for controlling movement or games.

How it works: Moving stick changes resistance on X and Y potentiometers. Center = ~2048, edges = 0 or 4095.

Key concepts: 2-axis analog input, potentiometer, coordinate mapping, game control

Range: X: 0-4095, Y: 0-4095, Button: 0/1

Try: Control a robot car's direction and speed using the joystick!

3.20 Acceleration Sensor

Component: SC7A20TR 3-axis Accelerometer

Measures acceleration force on 3 axes. Can detect tilt, shake, freefall, and orientation.

How it works: Tiny MEMS structure with proof mass. Acceleration causes capacitance change, measured by internal ADC.

Key concepts: G-force, 3-axis measurement, tilt detection, MEMS accelerometer

Range: ±2g to ±16g

Try: Build a step counter or a tilt-controlled game!

3.21 Potentiometer

Component: Rotary Potentiometer (10KΩ)

Variable resistor controlled by turning a knob. Output voltage proportional to knob position.

How it works: Wiper slides along resistive strip, changing the ratio of resistance and thus output voltage.

Key concepts: Variable resistance, analog input, voltage divider, user control

Range: 0-4095 (maps to 0V-3.3V)

Try: Control LED brightness or servo angle by turning the knob!

3.22 Reed Switch

Component: Reed Switch Module

Two metal strips in a glass tube that close when a magnet is nearby. Simple magnetic sensor.

How it works: Magnetic field magnetizes the reed contacts, pulling them together to close the circuit.

Key concepts: Magnetic field detection, digital input, proximity sensing

Output: LOW (0) = magnet near, HIGH (1) = no magnet

Try: Build a door/window alarm — triggers when magnet moves away!

3.23 Hall Sensor

Component: Hall Effect Sensor Module

Detects magnetic field strength. More sensitive and analog compared to reed switch.

How it works: Current-carrying conductor in magnetic field → Hall voltage proportional to field strength.

Key concepts: Hall effect, magnetic flux density, analog measurement

Range: 0-4095 analog value

Try: Build a magnetic field meter or detect hidden magnets!

3.24 Flame Sensor

Component: IR Flame Sensor Module

Detects fire/flame by sensing infrared light (760-1100nm wavelength) emitted by flames.

How it works: IR photodiode sensitive to flame spectrum. Analog output proportional to flame intensity.

Key concepts: Infrared detection, fire safety, threshold comparison, alarm systems

Detection angle: ~60°, range up to 80cm

Try: Build a fire alarm — buzzer sounds and LED flashes when flame detected!

3.25 Steam / Water Sensor

Component: Water/Steam Detection Module

Detects water, rain, or steam on its surface. The more water, the higher the reading.

How it works: Exposed traces on PCB. Water bridges traces, reducing resistance, increasing current.

Key concepts: Conductivity, moisture detection, analog measurement, weather sensing

Range: 0 (dry) to 4095 (fully wet)

Try: Build a rain detector — buzzer alarm when rain is detected!

3.26 UV Sensor

Component: GUVA-S12SD UV Sensor Module

Measures ultraviolet light intensity. Important for sun safety awareness.

How it works: Photodiode sensitive to UV-A/UV-B spectrum. Output voltage maps to UV index.

Key concepts: UV radiation, UV index scale, sun safety, environmental monitoring

UV Index: 0-2 (low), 3-5 (moderate), 6-7 (high), 8-10 (very high), 11+ (extreme)

Try: Build a sun safety alert — display warning when UV index is high!

3.27 Traffic Light Module

Component: 3-LED Traffic Light (Red/Yellow/Green)

Three colored LEDs mimicking a traffic signal. Each LED controlled independently.

How it works: Three digital output pins, each controlling one LED. Timing creates traffic patterns.

Key concepts: Sequential logic, timing, state machines, traffic control

Try: Program a realistic traffic light sequence with proper timing!

3.28 Relay Module

Component: 3V/5V Relay Module

An electronically controlled switch that can control high-power devices (lights, fans, motors).

How it works: Small current through coil creates magnetic field, pulling the switch contact to close/open the circuit.

Key concepts: Electromagnetic switching, isolation, high-power control, safety

Try: Control a lamp with a sensor — PIR triggers relay to turn on light!

3.29 Button Module

Component: Tactile Push Button Module

Simple push button with built-in pull-up resistor. Press to activate, release to deactivate.

How it works: Pressing the button connects the circuit. Pull-up resistor ensures clean HIGH/LOW signal.

Key concepts: Digital input, debouncing, pull-up resistor, event detection

Output: LOW (0) = pressed, HIGH (1) = released

Try: Make a button counter or use it to toggle LEDs on/off!

3.30 Tilt Sensor

Component: Ball Tilt Switch Module

Contains a small metal ball. When tilted, the ball rolls and closes/opens the circuit.

How it works: Metal ball rests on contacts when upright (circuit closed). Tilting moves ball away (circuit opens).

Key concepts: Tilt detection, mechanical switch, gravity, orientation sensing

Output: Digital HIGH/LOW based on tilt angle

Try: Build a tilt alarm — buzzer sounds when object is tilted!

3.31 Obstacle Avoidance Sensor

Component: IR Obstacle Avoidance Module

Detects nearby objects using infrared. The IR LED emits light, photodiode detects reflection.

How it works: IR LED transmits → if object is close, IR reflects back → photodiode detects it → output LOW.

Key concepts: IR reflection, proximity detection, obstacle avoidance, robotics

Range: 2-30cm (adjustable with onboard potentiometer)

Try: Build a robot that automatically avoids walls and obstacles!

3.32 Line Tracking Sensor

Component: IR Line Tracking Module

Detects black lines on white surface (or vice versa) using IR reflection difference.

How it works: Black surface absorbs IR (low reflection = HIGH output). White surface reflects IR (high reflection = LOW output).

Key concepts: IR reflection, line following, differential sensing, robot navigation

Positions: Left, Center, Right sensors

Try: Build a line-following robot that traces a path drawn on paper!

3.33 WiFi Connection

Component: ESP32 Built-in WiFi Module

ESP32 has built-in WiFi and Bluetooth. Connect to any WiFi network to send/receive data.

How it works: ESP32 WiFi radio connects to router using SSID and password. Gets IP address via DHCP.

Key concepts: WiFi protocol, SSID, IP address, DHCP, wireless networking

Standard: 802.11 b/g/n, 2.4GHz

Try: Connect to WiFi and print your device's IP address!

3.34 WiFi Real-time Display

Component: ESP32 WiFi + Web Dashboard

Send sensor data over WiFi to a web dashboard for real-time monitoring from any device.

How it works: ESP32 reads sensors → sends data via HTTP/WebSocket → web page displays graphs/values.

Key concepts: IoT (Internet of Things), HTTP requests, JSON data, web dashboard

Try: Build a remote temperature monitor viewable on your phone!

3.35 WiFi Remote Control

Component: ESP32 WiFi + Web Interface

Control your project remotely through a web interface! Send commands from phone or computer.

How it works: ESP32 runs a web server → browser sends commands → ESP32 executes (LED on/off, motor control, etc.)

Key concepts: Web server, HTTP GET/POST, remote control, client-server communication

Try: Control LEDs and buzzer from your phone's web browser!

Features: Auto lights (PIR + photoresistor), climate control (temp sensor + fan), security (RFID + servo lock), remote monitoring (WiFi)

Combined Project: Environmental monitoring system

Combines AHT20 (temp/humidity), BMP388 (pressure/altitude), UV sensor, and OLED display.

Features: Temperature, humidity, atmospheric pressure, altitude estimation, UV index — all displayed on OLED and sent via WiFi

Key concepts: Environmental sensing, data logging, multi-sensor fusion, weather forecasting

Try: Build a complete weather station and compare your readings with real weather reports!
