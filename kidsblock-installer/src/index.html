Welcome to AARNAITAI ROBO!

1

Drag Blocks

Pick blocks from the left toolbox and drag them into the workspace. Each block represents a programming command.

2

Connect Blocks

Snap blocks together like puzzle pieces to create your program. The order matters - programs run from top to bottom!

3

Run Your Code

Click the green "Run" button to execute your program. Watch the output appear in the console below!

4

Save & Share

Save your projects as .arobo files and share them with friends. You can also export code as JavaScript or Python!
